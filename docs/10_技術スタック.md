# 10 技術スタック

## 1. 技術スタック一覧

| レイヤー | 技術 | バージョン | 用途 |
|---|---|---|---|
| フロントエンド | Next.js | 14（App Router） | チャット UI・出典表示 |
| フロントエンド言語 | TypeScript | 5.x | 型安全なフロントエンド開発 |
| スタイリング | Tailwind CSS | 3.x | ユーティリティファーストの CSS |
| バックエンド | FastAPI | 0.110.x | REST API サーバー |
| バックエンド言語 | Python | 3.12 | バックエンド・バッチ処理 |
| ORM | asyncpg | 最新版 | PostgreSQL 接続 |
| データ検証 | Pydantic | v2 | リクエスト・レスポンスの型定義 |
| DB | Cloud SQL for PostgreSQL | 15 | ドキュメント・チャンク・履歴の永続化 |
| ベクトル検索 | pgvector | 0.7.x | コサイン類似度検索 |
| LLM | Vertex AI / Gemini API | gemini-2.5-flash | テキスト生成 |
| Embedding | Vertex AI / Gemini API | text-embedding-004 | ベクトル生成 |
| テキスト分割 | 独自 chunker | — | チャンク分割 |
| PDF パーサー | PyMuPDF（fitz） | 1.24.x | PDF テキスト抽出 |
| テストフレームワーク | pytest | 8.x | バックエンド・バッチのテスト |
| コンテナ | Docker / Docker Compose | 最新版 | ローカル開発・バッチ実行 |
| ホスティング（フロント） | Cloud Run | — | Next.js の配信 |
| ホスティング（バック） | Cloud Run | — | FastAPI の実行 |
| CI/CD | GitHub Actions | — | テスト自動実行 |
